# CImGui-Kinc

To git it:
`git clone --recursive https://github.com/Kode-Community/CImGui-Kinc.git`

To use as a lib, fork it.

Then to include your Kinc from somewhere else
- Win:
  ```
  git rm Kinc
  rmDir /s /q ".git/modules/Kinc"
  ```
- Unix's:
  ```
  git rm Kinc
  rm -rf .git/modules/Kinc
  ```
